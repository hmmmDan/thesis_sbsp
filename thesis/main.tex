\section*{Abstrakt}
Cílem této práce je navrhnout a implementovat herní mechaniky pro postavy videohry žánru 2D plošinovky. Práce se zaměřuje na systémy ovládání hráče a návrh logiky nepřítele. Mechaniky hráče nezahrnují pouze pohyb do stran, ale také systém skoku a střelby. Zvláštní důraz je kladen na implementaci skoku, jehož správné nastavení zásadně ovlivňuje herní pocit a odlišuje kvalitní 2D plošinovky od méně povedených titulů.

Součástí práce je také implementace fyzikálního systému. Herní engine Unity sice nabízí vestavěné nástroje pro práci s fyzikou, ty však nemusí vždy poskytovat požadovanou míru kontroly nad chováním postavy. Z tohoto důvodu jsou tyto nástroje využity především jako základ, zatímco pokročilejší herní mechaniky jsou řešeny vlastním kódem.

S programováním herních mechanik úzce souvisí také implementace animací. Po vytvoření grafických podkladů je nutné jednotlivé animace správně integrovat do enginu a propojit je s řídicí logikou tak, aby odpovídaly aktuálnímu stavu postavy.

Významnou část projektu tvoří rovněž systém střelby založený na zaměřování pomocí kurzoru myši. Tento přístup přináší řadu technických výzev, například správné určení směru projektilu, úpravu jeho rychlosti nebo zajištění jeho odstranění po určité době z důvodu optimalizace výkonu hry.

\textbf{Klíčová slova:} Unity, C\#, 2D plošinovka, herní mechaniky, fyzika, animace

\section*{Abstract}
The aim of this thesis is to design and implement game mechanics for a 2D platformer video game character. The work focuses on player control systems and enemy logic design. Player mechanics include not only lateral movement but also jumping and shooting systems. Special emphasis is placed on jump implementation, as its proper configuration significantly affects the game feel and distinguishes quality 2D platformers from lesser ones.

The thesis also includes the implementation of a physics system. Although the Unity game engine offers built-in tools for physics, they may not always provide the desired level of control over character behavior. Therefore, these tools are used primarily as a foundation, while more advanced game mechanics are implemented through custom code.

Programming game mechanics is closely related to animation implementation. After creating graphical assets, individual animations must be properly integrated into the engine and connected to the control logic to match the current state of the character.

A significant part of the project is also the shooting system based on aiming via mouse cursor. This approach brings several technical challenges, such as correctly determining projectile direction, adjusting its speed, or ensuring its removal after a certain time for game performance optimization.

\textbf{Key words:} Unity, C\#, 2D platformer, game mechanics, physics, animations

\section{Závěr}

V této práci jsem navrhl a implementoval herní mechaniky pro 2D plošinovku v Unity. Klíčovým prvkem bylo vytvoření komplexního systému pohybu hráče zahrnujícího pohyb, pokročilé skokové mechaniky (coyote time, jump buffer, wall jump, variable jump height) a mechaniku dashe.

Důležitým aspektem bylo také vytvoření vlastního fyzikálního systému namísto \newline spoléhání se na Unity vestavěné nástroje. Toto řešení umožňuje dynamickou úpravu gravitace podle aktuálního stavu hráče, což zásadně zlepšuje herní pocit.

Systém střelby byl navržen jako záchranný mechanismus s omezenou municí, což podporuje herní filosofii zaměřenou na zlepšování pohybových dovedností.

Bowlingová koule jako nepřítel poskytuje konstantní hrozbu s progresivní obtížností díky stun systému a růstu rychlosti.
